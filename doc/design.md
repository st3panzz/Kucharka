# Návrhová rozhodnutí

jazyk - PHP, HTML, CSS/ framework - Code Igniter, Bootstrap

databáze - MySQL Workbench 8.0 CE

vývojové prostředí a verze - Visual Studio Code - stabilní verze, Atom - stabilní verze
